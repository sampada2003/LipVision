To Run LipVision Webapp follow below steps:
1. Clone the Whole Repository.
2. Open Command Prompt.
3. Navigate to the cloned folder, and run **python -m streamlit run streamlitapp.py**
4. The Webapp would work and will look something like this
   <img width="1920" height="1417" alt="image" src="https://github.com/user-attachments/assets/747ab52a-3cb0-4956-bc85-693a78e465f3" />
