To Run LipVision Webapp follow below steps:
1. Clone the Whole Repository.
2. Open Command Prompt.
3. Navigate to the cloned folder using cd in command prompt:
   
cd <Your Preffered Download Directory>/LipVision-main/app
run **python -m streamlit run streamlitapp.py** 

5. The Webapp will run in your browser and will look something like this
   <img width="1920" height="1417" alt="image" src="https://github.com/user-attachments/assets/747ab52a-3cb0-4956-bc85-693a78e465f3" />

6. Select different videos from the dropdown and the sentence spoken in it will be transformed into text
   <img width="309" height="293" alt="image" src="https://github.com/user-attachments/assets/344bb299-ed40-4f65-951e-7e3959e20343" />

**NOTE:** If there are any error importing the libraries please refer to Libraries.txt file to view list of libraries along with its versions used while developing the model.
